## Custom Retail Payment Analytics Project

This customized streaming analytics project extends the original Kafka consumer example by adding retail payment transaction analytics. The project tracks payment methods such as PayPal, Apple Pay, and credit card transactions in real time while processing streaming sales data.

### Custom Modifications

- Created a custom Kafka consumer file named `kafka_consumer_kiruthikaa.py`
- Added payment method tracking logic
- Generated a new `payment_summary.csv` output artifact
- Logged real-time payment analytics summaries during message consumption

### Business Value

This enhancement simulates a real-world omnichannel retail analytics workflow where payment transaction summaries are monitored for operational insights, reporting, and fraud analysis.
